# Mindful Muslim – Salah Habit Tracker

Mindful Muslim is a mobile application designed to help Muslims maintain consistent salah (prayer) habits. The app focuses on supporting students at the International Islamic University Malaysia (IIUM) by providing tools to track prayer performance, receive reminders, and monitor spiritual progress.

## Project Overview

Many students struggle to consistently perform the five daily prayers due to busy schedules and distractions. Mindful Muslim addresses this issue by providing a centralized platform to log prayers, track habits, and encourage spiritual discipline. :contentReference[oaicite:0]{index=0}

The application promotes accountability, motivation, and community engagement to help users strengthen their connection with their faith.

## Key Features

- Prayer tracking system (On time, Late, Missed, Jamaah)
- Prayer reminders and motivational hadith
- Weekly and monthly prayer analytics
- Friend tracking to encourage consistency
- Invite friends to track prayer habits together
- User profile and feedback system

## Target Users

The primary users of this application are **IIUM students**, including both undergraduate and postgraduate students who want to track their salah performance and improve their spiritual discipline. :contentReference[oaicite:1]{index=1}

## Tech Stack

- React Native
- Expo
- Firebase (Authentication & Firestore)
- TypeScript
- Visual Studio Code
- Figma
- Lucidchart

Firebase is used to support real-time data synchronization, authentication, and analytics for the application. :contentReference[oaicite:2]{index=2}

## System Features

The application allows users to:

1. Register or log in to the system
2. Update prayer status
3. Receive reminders for missed prayers
4. View weekly and monthly prayer analytics
5. Track prayer progress with friends
6. Update profile and submit feedback

## Development Methodology

The project uses a **hybrid approach combining Waterfall and Agile**:

- Waterfall for structured planning, documentation, and academic milestones
- Agile for coding, testing, and iterative improvements

This combination allows systematic development while enabling continuous feature improvements.

## System Architecture

The database design includes several core entities:

- User
- Prayer Log
- Venue
- Analytics
- Friend
- Settings
- Feedback

These entities support prayer tracking, analytics, and social interaction within the application.

## Results

The Mindful Muslim application successfully provides a structured system for tracking daily prayers and spiritual habits. Initial testing with IIUM students showed that the app helps users monitor their prayer consistency and encourages discipline in worship. :contentReference[oaicite:3]{index=3}

## Limitations

- Currently available only on Android
- Google Maps API integration increases operational costs

These limitations will be addressed in future improvements.
